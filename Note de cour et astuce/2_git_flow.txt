

Type de branche 	Environnement cible
hotfix 				Correction de bug
feature 			Local ou environnement éphémère (Docker)
develop 			Intégration
release 			Qualification / Recette
master 				Production


git flow init
-> develop
-> feature
-> bugfix
-> hotfix
-> support
-> ENTER
-> ENTER
