1) Lire la documentation de Symfony sur la traduction :
https://symfony.com/doc/current/translation.html

 

2) Créer une nouvelle branche avec git flow :
git flow feature start translation

 

3) Configurer Symfony pour que, par défaut, la langue française soit affichée

 

4) Modifier les templates pour être capable d'afficher les traductions

 

5) Générer le fichier de traduction de la langue française

 

6) Traduire les messages dans le fichier de traduction de la langue française

 

7) Vérifier (et éventuellement corriger) le bon affichage des textes en français

 


8) Intégrer ces modifications de code dans la nouvelle branche :
-----------------------------
git checkout feature/translation
git add .
git commit -m "[Translation] Add the French language for category screens #1"
git flow feature publish translation
-----------------------------

 

9) Et enfin, publier ce travail :
######################################################################
git flow feature finish translation
git push
